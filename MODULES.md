# Module manifest

Folder names and order mirror the official Microsoft Learn course page for
AI-103 ("Prepare for the exam" → 4 learning paths, in this order, each
listing its modules in study order). Base URL prefix for module links:
`https://learn.microsoft.com/en-us/training/modules/`
